CSS속성 선택자

기본 선택자 뒤에[]를 사용하여 특정 속성과 해당 속성의 값을 통해 스타일을 설정할때 사용한다.

            선택자[속성=속성값]{...}형식으로 쓰인다
            -- 기호 별 의미 --
            = : 속성의 값이 같을때
            ~= : 특정 단어를 속성값으로 가질 때(띄어쓰기된 이름 기준 구분)
            EX) id~=test ---> id="test abcd qwer" ---- O
                          --> id="test1234" .... X
            |= : 특정 단어를 속성값으로 가질 때(""안에 포함되는 것 모두 확인)
            ^= : 속성 값이 특정 단어로 끝날 때
            *= : 속성값이 특정 단어를 포함할 때

도위 선택자

동위 선택자는 동급관계에 있는 형제 요소에 스타일을 설정할때 사용한다. 

            선택자A+선택자B {...} : A바로 뒤에 잇는 B 요소를 선택할때
            선택자A~선택자B {...} : A뒤의 모든 B요소를 선택할때

반응 선택자

사용자의 움직임(마우스 클릭,마우스를 올려 놓았을때)을 감지하여 스타일을 변경하는 선택자이다. 

            선택자 :active{...} / 사영자가 마우스를 클릭했을때
            선택자 :hover{...} / 마우스 커서가 해당 요소 위에 올라갔을때

상태 선택자

입력형식에 따라 변화하는 스타일을 설정할때 사용한다 

            input:checked{...} / 체그 상태 여부에 따른 스타일

focus(촛점)가 맞춰진 상태에서 스타일 적용도 가능하다.


사용 가능 여부와 관련한 스타닝ㅇ 적용도 가능하다.
